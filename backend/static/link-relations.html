Link Relations

Relation Name Description
users-all Refers to the list of all existing users
add-user Refers to a resource which can be used to create a new user
movies-all Refers to the list of all existing movie
add-movie Refers to a resource which can be used to create a new movie
categories-all Refers to the list of all existing categories
add-category Refers to a resource which can be used to create a new category
reviews-of-user Refers to the list of all reviews written by a specific user
reviews-for-movie Refers to the list of all reviews written for a specific movie
add-review Refers to a resource which can be used to create a new review for a specific movie
delete Refers to a resource that can be used to delete the link's context.
